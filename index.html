Generative Art Synthesizer

A harness and control panel for a suite of generative drawing algorithms, with live parameter shaping and modulation.

Triple Slider Anatomy

Noise speed button: cycles modulation level from 0 to 5 bars.

Slider track: drag handles horizontally to set values.

Min bound: lower limit of the modulation range.

Current value: the active value used by the algorithm.

Max bound: upper limit of the modulation range.

These sliders shape algorithm parameters in real time. Try widening ranges, shifting current values, and changing modulation speed to see how each setting transforms the generated artwork.
